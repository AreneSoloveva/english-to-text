# English-to-text
# Распознай японский текст с изображения!

Web-приложение для распознавания текста на английском/немеком/французском/испанском языках с изображения.
Используются библиотеки:

- [Streamlit](https://streamlit.io/)
- [Transformers]( https://huggingface.co/)
- [Torchvision](https://pytorch.org/vision/stable/index.html)

Для распознавания изображений используется нейронная сеть [Manga OCR](https://huggingface.co/kha-white/manga-ocr-base). 

[Ссылка на развернутое приложение](https://english-to-text-bgr3vznabufbszvctrdutf.streamlit.app/).
